# OpenClaw AI Chat for Android 🤖
*(Scroll down for English version)*

**OpenClaw AI Chat** คือแอปพลิเคชัน Android ที่ออกแบบมาเพื่อให้ผู้ใช้สามารถใช้งาน AI Agent ส่วนตัวบนโทรศัพท์มือถือได้อย่างยืดหยุ่น โดยสามารถปรับเปลี่ยนโมเดล AI ได้หลากหลายค่าย (เช่น OpenAI, Anthropic, Google, Mistral, Ollama) และมีความสามารถพิเศษในการ "จดจำเรื่องราว (Long-term Memory)" ของผู้ใช้ได้ข้ามเซสชัน

## ✨ ฟีเจอร์หลัก (Features)
- 🌐 **รองรับหลายผู้ให้บริการ AI**: ใช้งาน API ได้หลายค่าย (OpenAI, Anthropic, Google, Groq ฯลฯ)
- 🗣️ **ระบบหลายภาษา (Localization)**: รองรับเมนูและการแสดงผลถึง 5 ภาษา (ไทย, อังกฤษ, สเปน, ญี่ปุ่น, จีน)
- 🎨 **ปรับแต่งหน้าตาได้ตามใจ (Customization)**: 
  - สลับโหมด Light/Dark ได้อิสระ
  - ชุดสี (Color Palette) ถึง 4 แบบ: Neon, Ocean, Forest, Sunset
  - ปรับขนาดตัวอักษรของแชตได้
- 🧠 **ความทรงจำระยะยาว (Long-term Memory)**: AI สามารถจดจำข้อมูลสำคัญจากบทสนทนาเก่าและนำมาอ้างอิงได้
- 🔒 **ทำงานแบบออฟไลน์/จัดเก็บข้อมูลในเครื่อง**: ไฟล์ความทรงจำและประวัติการสนทนาถูกจัดเก็บไว้ในตัวเครื่องของคุณเท่านั้น ปลอดภัยจากการดักจับข้อมูล

## 🛠️ วิธีการติดตั้งและใช้งาน (How to Use)
1. **ติดตั้งแอปพลิเคชัน:** ดาวน์โหลดและติดตั้งไฟล์ `app-debug.apk` บนโทรศัพท์ Android ของคุณ
2. **การตั้งค่า API Key:** 
   - เปิดแอปพลิเคชัน แตะไอคอนรูปเฟือง (Settings) ที่มุมขวาบน
   - เลือกผู้ให้บริการ AI ที่ต้องการใช้งาน
   - ใส่ `API Key` ของคุณลงในช่อง (สามารถกดปุ่ม "Get API Key" เพื่อเปิดหน้าเว็บสำหรับสมัครขอรับ Key ได้)
3. **ปรับแต่งและใช้งาน:** 
   - เมื่อตั้งค่าเสร็จสิ้น กด "Save Settings" และเริ่มพูดคุยกับ AI ได้ทันที!
   - หากต้องการล้างความทรงจำ สามารถเข้าไปกดที่ "Clear All Memories" ได้ในหน้าตั้งค่า

# OpenClaw AI Chat for Android 🤖
*(English Version)*

**OpenClaw AI Chat** is an open-source Android application designed to let you host a personal AI Agent right on your mobile device. It provides maximum flexibility, allowing you to switch between various AI models (e.g., OpenAI, Anthropic, Google, Mistral, Ollama) while utilizing long-term memory across chat sessions.

## ✨ Features
- 🌐 **Multi-Provider Support**: Connects to multiple AI API providers out of the box.
- 🗣️ **Localization**: Supports 5 languages natively (English, Thai, Spanish, Japanese, Chinese).
- 🎨 **Deep Customization**: 
  - Light/Dark mode toggles.
  - 4 distinct color palettes: Neon, Ocean, Forest, Sunset.
  - Adjustable chat font sizes.
- 🧠 **Long-term Memory**: The AI Agent can recall important facts and contexts from past conversations.
- 🔒 **Local Storage**: All conversation history and memories are securely stored locally on your device.

## 🛠️ How to Use
1. **Install the App:** Download and install the `app-debug.apk` onto your Android phone.
2. **Setup API Key:**
   - Launch the app and tap the gear icon (Settings) on the top right.
   - Choose your preferred AI Provider.
   - Enter your `API Key` (You can tap the "Get API Key" button to get directed to the provider's registration page).
3. **Customize & Chat:**
   - Click "Save Settings" and start chatting with your AI!
   - To clear your data, use the "Clear All Memories" button located in the settings menu.
